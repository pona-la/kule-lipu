# kule-lipu

## Explanation of the repo structure

```ruby
.
├── assets # All of the site's assets are stored here
│   ├── css # stylesheets
│   │   ├── kule.scss # The primary stylesheet
│   │   └── kule-tu.css # Per site stylesheet
│   └── sitelen # images
├── _includes # Parts of the pages that get reused
│   ├── lawa.html # Head section of html
│   ├── nena.html # Navigation at the top of the pages
│   └── noka.html # Footer on the bottom of the pages
├── _layouts # Page layouts
│   ├── default.html # Basis of all the layouts
│   ├── lipu.html # page with lawa and nena from includes
│   └── pona.html # simple layout without anything
└── _sass
    └── kule.scss # The primary stylesheet source file

```
